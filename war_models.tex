\subsection[Модель Осипова~--- Ланчестера]
           {Модель боевых действий Осипова~--- Ланчестера}  \label{war_Osipov}

 Эта модель\footnote{Модель была предложена в 1915 году русским офицером
                     М.\,П.~Осиповым и, независимо, в 1916~году английским ученым и
                     инженером Ф.~Ланчестером (F.~Lanchester).}
  применяется для описания динамики военных сражений. Пусть $B(t)$~--- численность
 армии <<синих>> в момент времени $t$, $R(t)$~--- численность армии <<красных>>.
 Средняя эффективность стрельбы <<синих>> и <<красных>> задаётся константами $b$
 и $r$ соответственно. Например, если $r = 0{,}7$, то в среднем из каждых $10$
 выстрелов <<красных>> $7$ поражают солдат противника. Таким образом, параметры
 $b$ и $r$ характеризуют технические характеристики оружия и обученность солдат,
 его использующих, а таже тактическую грамотность и опыт командиров.

 Построим модель боевых действий между армиями <<синих>> и <<красных>> с учетом
 следующих упрощающих предположений:
 \begin{enumerate}
   \item [---] противники несут потери только от вражеского огня;
   \item [---] подкреплений не поступает ни к одной из сторон;
   \item [---] <<победа>> означает полное уничтожение противника при ненулевой
               численности собственной армии;
   \item [---] возможна патовая ситуация~--- <<ничья>>, положение, приводящее к взаимному
               уничтожению, когда $B(t) \rightarrow 0$, $R(t) \rightarrow 0$.
 \end{enumerate}

 Модель, с учётом имеющихся упрощений, описывается системой дифференциальных уравнений
%%------------------------------------------------------------------------------
\begin{eqnarray}
 \dot{R}\!&=&\!-b B; \nonumber \\
 \dot{B}\!&=&\!-r R; \nonumber \\
   B(0) \!&=&\! B_0, \quad R(0) = R_0. \nonumber
\end{eqnarray}
%%------------------------------------------------------------------------------

Решение имеет вид
%%------------------------------------------------------------------------------
\begin{eqnarray}
 B\!&=\!&C_1  e^{ t\sqrt{br} } + C_2 e^{ -t\sqrt{br} }; \nonumber \\
 R\!&=\!&-C_1 \sqrt{\frac{b}{r}}\,e^{ t\sqrt{br} } +
        C_2 \sqrt{\frac{b}{r}}\,e^{ -t\sqrt{br} }. \nonumber
\end{eqnarray}
%%------------------------------------------------------------------------------
Константы интегрирования определяются из начальных условий:
$$%%----------------------------------------------------------------------------
 C_1 = \frac{1}{2} \left( B_0 - R_0 \sqrt{ \frac{r}{b} } \; \right), \qquad
 C_2 = \frac{1}{2} \left( R_0 \sqrt{ \frac{r}{b} } + B_0\right).
$$%%----------------------------------------------------------------------------
Отсюда, в патовой ситуации $C_1 = 0$ и, следовательно, выполняется \emph{закон Ланчестера:}
$$%%----------------------------------------------------------------------------
  B_0 = R_0 \sqrt{ \frac{ \, r \,}{b} }.
$$%%----------------------------------------------------------------------------
Например, чтобы поставить противника, имеющего 3-хкратный численный
 перевес, в патовую ситуацию нужно иметь в 9 раз более эффективное оружие
 (и\slashили лучше обученный личный состав).

Таким образом, получаем, что
$$%%----------------------------------------------------------------------------
  B = C_2 \, e^{ -t\sqrt{br} }, \qquad
  R = C_2  \sqrt{\frac{b}{r}} \, e^{ -t\sqrt{br} }
$$%%----------------------------------------------------------------------------
есть убывающие величины одного порядка.

\question\label{war_Osipov_vopros_1} Пользуясь моделью Ланчестера~--- Осипова проанализировать
бой группы сильнее бронированных, но менее скорострельных танков (советские ИС-2) против менее
защищенных, но более скорострельных танков (немецкие Тигры PzVI). Принять скорострельность ИС
равной $3$ выстрела в минуту, Тигра "--- $9$ выстрелов в минуту, вероятность поражения ИС равной
$0{,}1$, Тигра "--- $0{,}9$, начальное количество равно $20$ машинам у каждого противника.

 По условию игры перевал

 \begin{enumerate}
   \item  Батальоны направить на различные перевалы (всё равно

   \item  Оба батальона штурмуют один перевал (любой из четырёх).
 \end{enumerate}

 \begin{enumerate}
   \item  Поставить по $1$-му батальону на каждый из перевалов;
   \item  Направить по $2$ батальона на два перевала, остальные

   \item  На один из перевалов поставить $2$~батальона, ещё каждый из
          двух перевалов защищать силами $1$-го батальона и один перевал
          оставить без защиты.
 \end{enumerate}
 Остальные возможные стратегии <<Красных>> явно абсурдны и потому далее не
 рассматриваются.

Проведём количественную оценку выигрышей во всех возможных
 ситуациях игры. В ситуации $(1, 1)$ ($1$-я стратегия <<Синих>> и $1$-я стратегия

 $5/6$. $(1, 3)$~--- выигрыш <<Синих>> $= 1/2$ (всего есть $6 = C_4^2$ различных
 вариантов для <<синих>>, из которых проигрышных $3 = C_3^2$). В
 остальных ситуациях вероятности победы <<Синих>> вполне очевидны.
